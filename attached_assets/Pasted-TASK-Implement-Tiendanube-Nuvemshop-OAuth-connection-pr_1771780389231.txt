REFERENCE (MUST FOLLOW):
Tiendanube OAuth flow:
1) Redirect user to: https://www.tiendanube.com/apps/{client_id}/authorize?state=...
2) Callback receives ?code=...&state=...
3) Exchange code for token via POST https://www.tiendanube.com/apps/authorize/token with JSON:
   { "client_id": "...", "client_secret":"...", "grant_type":"authorization_code", "code":"..." }
Source: https://tiendanube.github.io/api-documentation/authentication

API Base URL: https://api.tiendanube.com/v1
Product resource docs: https://tiendanube.github.io/api-documentation/resources/product
